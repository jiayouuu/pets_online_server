宠物领养系统开发文档
一、系统概述
1. 目标
为用户提供宠物领养全流程服务，包括浏览、申请、互动及物流跟踪。

为管理员提供高效管理工具，覆盖用户、宠物、审核及数据分析。

扩展功能：在线问诊、用户自主发布领养信息（社区论坛）、物流跟踪。

二、技术架构
后端架构（Spring Boot3）
模块	技术选型	说明
安全框架	Spring Security + JWT	权限控制与身份认证
数据库	MySQL 8.x + Redis	主数据存储 + 缓存
ORM框架	MyBatis-Plus	简化SQL操作
文件存储	图片/视频存储
三、详细功能模块设计
用户端功能
1. 用户模块
注册/登录

邮箱验证

密码加密存储（BCrypt算法）

个人中心
头像，昵称，密码修改，用户收藏，用户点赞，
领养申请历史查询（按状态筛选）

2. 宠物模块
宠物信息展示

列表页：按品种、年龄、地理位置筛选

详情页：宠物图片轮播、健康状态、性格描述

物流跟踪：

用户发布领养信息（社区论坛扩展）

发布领养帖：填写宠物信息 + 上传图片

帖子审核机制（识别到敏感词后交由管理员复核，否则直接发布）
帖子被用户举报后也交由管理员复核
用户互动：评论、私信、点赞、收藏、举报

3. 领养流程
在线申请
填写申请表（家庭环境、养宠经验）
电子签名（Canvas手写签名 + 防篡改）

进度查询
状态流转：提交申请 → 审核中  → 物流中转→ 完成领养

4. 在线问诊（扩展功能）


图文咨询
用户上传宠物症状图片/视频

宠物医生回复（支持富文本 + 处方模板）

5. 社区论坛
板块分类

领养信息发布（用户自主发布）
养宠经验交流（帖子 + 评论）
宠物求助专区（紧急领养/医疗援助）
内容安全

敏感词过滤（AC自动机算法 + 动态词库）

管理端功能
1. 用户管理
用户行为分析（登录频率、发帖数量）
封禁违规账号（连带删除敏感帖子）

2. 宠物管理
领养信息审核（社区论坛扩展）
审核用户发布的领养帖（真实性校验）

3. 数据统计
用户，帖子等数据统计


注意：整个系统重要节点信息使用邮件推送
